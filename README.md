Command line utilities for interfacing with Xirvik.

## Installation

### Poetry

```shell
poetry add xirvik-tools
```

### Pip

```shell
pip install xirvik-tools
```

## Usage

Add `-d` to show debug logs.

```shell
xirvik-tools
```
